# FreeBie
FreeBie - адаптивный кроссбраузерный макет сайта
## Стек технологий

+ HTML5
+ SASS
+ GULP
+ JAVASCRIPT
+ SVG

## Установка

+ Клонировать проект

```
    git clone https://github.com/GaynetdinovR/FreeBie.git
```

+ Установить необходимые пакеты

```
    npm install
```

+ Запустить таск-менеджер

```
    gulp
```
