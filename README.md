## Установка и запуск
<a id="getting_started"></a>

- Клонировать проект

```
git clone https://github.com/GaynetdinovR/FreeBie.git
```

- Установить необходимые пакеты

```
npm install
```

- Запустить локальный сервер

```
npm run dev
```

- Открыть в браузере

```
http://localhost:3000/
```
